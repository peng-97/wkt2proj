


#  wkttoproj

## Intro



> 将坐标字符串转为proj格式
> 


## Api

* **to_proj4(wkt)**


## Usage

```javascript
 import {to_proj4} from 'wkt2proj';
   
let result=to_proj4(wkt)
```
  

## Html use CDN   
```javascript

<script src='https://unpkg.com/wkt2proj/dist/wkt2proj.min.js' ></script>
<script>
    var result=wkt2proj.to_proj4(wkt)
</script>

```
